<Server copy.py程序分析 >
程序简介：
    这是一个基于Flask框架搭建的服务器程序。它的主要作用是向远程设备发送控制指令，并通过消息队列接收设备返回的数据。开发者可以用它管理物联网设备或分布式计算节点。
    
核心功能模块
    指令发送功能:
        访问路径：/send-command
        接收POST请求
        请求需要包含JSON格式的指令内容
        程序会把指令推送到AWS的消息队列

    响应接收功能:
        访问路径：/receive-response
        处理GET请求
        从另一个消息队列读取设备返回的数据
        返回JSON格式的响应内容
适用场景:
    物联网设备管理
    自动化测试
    分布式任务调度
